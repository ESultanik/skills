# esultanik-skills

[Evan Sultanik](https://www.sultanik.com/)'s personal Claude Code skills marketplace.

## Quick Start

### Add the Marketplace

```
/plugin marketplace add esultanik/skills
```

### Install a Plugin

```
/plugin install interslavic@esultanik-skills
```

## Available Plugins

| Plugin | Description | Skills |
|--------|-------------|--------|
| [interslavic](plugins/interslavic/) | Translation to/from Interslavic | `interslavic:translate` |
