# Interslavic Translation Plugin

A Claude Code plugin for translating between English, major Slavic languages, and Interslavic.

## Supported Languages

| Code | Language |
|------|----------|
| ISV | Interslavic |
| EN | English |
| RU | Russian |
| BE | Belarusian |
| UK | Ukrainian |
| PL | Polish |
| CS | Czech |
| SK | Slovak |
| SL | Slovenian |
| HR | Croatian |
| SR | Serbian |
| MK | Macedonian |
| BG | Bulgarian |
| CU | Old Church Slavonic |
| DE | German |
| NL | Dutch |
| EO | Esperanto |

## Installation

```
/plugin marketplace add esultanik/skills
/plugin install interslavic@esultanik-skills
```

## Skills

### translate

Translate text between supported languages.
