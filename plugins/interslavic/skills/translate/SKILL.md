# Interslavic Translation Skill

This skill translates text between English, major Slavic languages, and Interslavic.

## Supported Languages

- **English** (EN)
- **Russian** (RU)
- **Ukrainian** (UK)
- **Polish** (PL)
- **Croatian** (HR)
- **Czech** (CZ)
- **Bulgarian** (BG)
- **Interslavic** (ISV)

## Usage

When asked to translate text, identify the source language and target language(s), then provide the translation.
